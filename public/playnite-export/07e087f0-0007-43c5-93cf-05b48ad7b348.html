Description

Daily OVR Free is a free version of Daily OVR. It provides minimum essential overlay features. We think it would benefit a lot of VR users! :)
If you like Daily OVR Free and want to use advanced features, you can consider purchasing Daily OVR.

What is Daily OVR?


Do you want to use your desktop/window apps during VR gameplay? For example, you can play VRChat and see your Twitch chat window. Daily OVR is the best SteamVR Overlay(OVR) solution! And it supports Oculus, Vive/Viveport/Cosmos/MR(Mixed Reality)/Index!

Differences between Free and Paid


  • Free version only supports up to 2 opened overlays. (Paid version can support more numbers)
  • Limited numbers (2) of switch programs on the QuickSwitch panel. If you want to switch more programs, you have to close the overlay and reopen it.
  • Only one entry of My Favorites
  • Only one CustomApp
  • You can only bind an overlay to the RIGHT controller. (cannot bind to the left controller, HMD, tracker)

Daily OVR doesn't work in some games. For example, Elite dangerous, PokerStarVR ...etc


The issue is caused by Elite Dangerous tries to run in Oculus SDK mode, and Daily OVR doesn't support Oculus SDK. There is a workaround to force it to run in SteamVR mode, but it doesn't work every game (ED, PokerStarVR are OK). You may have controller issues. The possible problem is the game developer didn't think that users will forcibly use SteamVR to play the game; they didn't design the controller mapping correctly. You can search "daily ovr q&a" on Google to see the steps.

How many overlays can I open?


You can open up to 5 overlays. But it depends on your PC's performance. If performance is terrible, you could try:
  • Close the overlays
  • Set lower FPS value


The button mappings of controllers are not good. Can I change it?


Daily OVR supports SteamVR 2.0 Input. You can customize button settings yourself.

I cannot see/move my mouse pointer on some apps!


Some apps run as administrator permission. When Daily OVR doesn't run as the same permission, it cannot access to those apps.
You have to check "Run this program as an administrator" on DailyVRLauncher.exe Properties.
